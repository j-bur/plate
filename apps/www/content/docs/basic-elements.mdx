## Features

- Includes commonly used block elements: blockquote, code block, headings, and paragraph.

</PackageInfo>

**Plugins:**

- **`createBlockquotePlugin()`** for `blockquote` element
- **`createCodeBlockPlugin()`** for `code_block` element
- **`createHeadingPlugin()`** for `h1`, `h2`,... elements
- **`createParagraphPlugin()`** for `p` element

## Installation

```bash
npm install @udecode/plate-basic-elements
# or
npm install @udecode/plate-block-quote @udecode/plate-code-block @udecode/plate-heading @udecode/plate-paragraph
```

## Usage

```tsx
// ...
import { createBasicElementsPlugin } from '@udecode/plate-basic-elements';

const plugins = [
  // ...otherPlugins,
  createBasicElementsPlugin(),
];
```

### Options

```tsx
export type HotkeyPlugin = {
  /**
   * Hotkeys to listen to trigger a plugin action.
   */
  hotkey?: string | string[];
};

export interface CodeBlockPlugin extends HotkeyPlugin {
  syntax?: boolean;
  syntaxPopularFirst?: boolean;
  deserializers?: string[];
}

export interface HeadingPlugin extends HotkeyPlugin {}

export interface HeadingsPlugin {
  /**
   * Heading levels supported from 1 to `levels`
   */
  levels?: number;
}
```

## Keyboard Interactions

<KeyTable>
  <KeyTableItem hotkey="⌘ + ⌥ + 0">
    Toggle between a paragraph and normal text formatting.
  </KeyTableItem>
  <KeyTableItem hotkey="⌘ + ⌥ + 1">
    Toggle between a paragraph and normal text formatting.
  </KeyTableItem>
</KeyTable>

- `mod+shift+.`: Toggle the blockquote formatting for selected text.
- `mod+opt+8', 'mod+shift+8`: Toggle the code block formatting for selected text.
- `mod+a`: Select all text in the editor.
