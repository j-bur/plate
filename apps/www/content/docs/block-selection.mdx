## Features

- Select entire blocks, as opposed to individual words or characters.
- To select an entire block, mouse down outside the text area and then move the cursor into the block. Once it is selected, you'll see a background color.
- Keep moving down or up to select multiple blocks.
- Once selected, the available actions are: copy, cut, and delete.

## Installation

```bash
npm install @udecode/plate-selection
# or
npm install @udecode/plate-node-id
```

## Usage

```tsx
import { createNodeIdPlugin } from '@udecode/plate';
import { createBlockSelectionPlugin } from '@udecode/plate-selection';

const plugins = createPlugins(
  [
    // ...otherPlugins,
    createNodeIdPlugin(),
    createBlockSelectionPlugin(),
  ],
  {
    components: createPlateUI(),
  }
);
```

Note: the Selection plugin needs `id` property on each selectable block. You can use `createNodeIdPlugin()` to add it automatically.

### Options

```tsx
export interface BlockSelectionPlugin {
  query?: QueryNodeOptions;
}
```

### `BlockSelectionPlugin`

A plugin used to manage block selection within the editor.

<APIParameters>
  <APIItem name="query" type="QueryNodeOptions" optional>
    Options for querying nodes within the editor.
  </APIItem>
  <APIItem name="onKeyDownSelecting" type="(e: KeyboardEvent) => void" optional>
    A function that gets called when a key is pressed while a block is selected.
  </APIItem>
  <APIItem
    name="sizes"
    type="{ left?: number; top?: number; bottom?: number; right?: number; }"
    optional
  >
    An object specifying the block's dimensions from each side. Values are in
    pixels.
  </APIItem>
</APIParameters>
