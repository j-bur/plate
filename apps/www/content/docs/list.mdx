### Features

The List plugin allows you to create both ordered (numbered) and unordered (bulleted) lists within your Plate editor.

### Installation

```bash
npm install @udecode/plate-list
```

### Usage

```tsx
import { createListPlugin, createTodoListPlugin } from '@udecode/plate';

const plugins = createPlugins(
  [
    // ...otherPlugins,
    createListPlugin(),
  ],
  {
    components: createPlateUI(),
  }
);
```

#### Options

```tsx
export interface ListPlugin extends HotkeyPlugin {
  /**
   * Valid children types for list items, in addition to p and ul types.
   */
  validLiChildrenTypes?: string[];
}
```

### Keyboard Interactions

- `mod+opt+4, mod+shift+4`: Insert a numbered list.

## Todo List

<ComponentPreview name="playground-demo" id="list" />

### Installation

```bash
npm install @udecode/plate-list
```

### Usage

```tsx
import { createTodoListPlugin } from '@udecode/plate';

const plugins = createPlugins(
  [
    // ...otherPlugins,
    createTodoListPlugin(),
  ],
  {
    components: createPlateUI(),
  }
);
```

#### Options

```tsx
export interface TodoListPlugin extends HotkeyPlugin {
  inheritCheckStateOnLineStartBreak?: boolean;
  inheritCheckStateOnLineEndBreak?: boolean;
}
```

`inheritCheckStateOnLineStartBreak`: This setting is triggered when a todo item line is broken from the beginning.
It takes on the checkbox state of the current todo item when a new one is made above it.
By default, its value is set to `false`.

`inheritCheckStateOnLineEndBreak`: This setting is triggered when a todo item line is broken from the end.
It takes on the checkbox state of the current todo item when a new one is made below it.
By default, its value is set to `false`.

### Keyboard Interactions

- `mod+opt+4, mod+shift+4`: Insert a numbered list.
- `backspace` : Delete backward.
